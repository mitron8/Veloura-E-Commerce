# 🛍️ Veloura – Full Stack E-Commerce Website

Veloura is a fully functional **MERN Stack E-Commerce Web Application** where users can browse products, add items to cart, and place orders seamlessly.  
The project is built with a focus on clean UI, scalability, and real-world functionality.

---

## 🚀 Live Demo
🔗 https://veloura-e-commerce-frontend-frqs.onrender.com/

---

## 📌 Features

- 🛒 Add to Cart & Remove from Cart
- 🔄 Update Product Quantity
- 👤 User Authentication (Login/Register)
- 📦 Order & Checkout System
- 📱 Responsive Design (Mobile + Desktop)
- ⚡ Fast API with Express & MongoDB
- 🔐 Secure Backend with JWT/Auth

---
